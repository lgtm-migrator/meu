## Usage
### as Module
install
```shell
npm i meu
```
***.js
```javascript
import meu from 'meu'

console.log(meu(つらい))
// console->「つらいめう」
```
### as Shell
install
```shell
$ npm i -g meu
# (省略)

$ meu めう、めうめう
めう、めうめうめう！

$ meu つらい
つらいめう！

$ meu お金ほしい 時間ほしい とてもつらい
お金ほしいめう！
時間ほしいめう！
とてもつらいめう！
```
